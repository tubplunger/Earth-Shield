Looking back at the state of Earth Shield, right now it is in a good spot though some things still need to be added in and ironed out.

The architecture of the game holds up really well. Each script is doing its own thing and, while some of them are connected to get things like scoring working, they are in charge of something unique to them rather than being crammed into a few scripts or something.

None of it was particularly challenging to do. I say the most challenging thing was getting screen wrapping going. Start the screen wrapping wasn't an issue and it worked then, though with asteroid spawning being set just outside of the screen the screen wrapping would catch it. That took a little bit of tweaking so that the asteroids weren't caught on the way to the playable area.

For some stretch goals, I already did screen wrapping so that can be taken off the list. I know I want to do Asteroid Splitting, it is something I keep mentioning in my reports, multiple asteroid sizes, animated space background, and some basic particle effects. I think all of these would be interesting to try.
